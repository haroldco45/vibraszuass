# 🛵 Vibras Zuass! - Domicilios Caucasia

Plataforma integral para la gestión de domicilios en Caucasia, Antioquia. 
Conecta clientes, restaurantes y domiciliarios en tiempo real.

### 📁 Estructura del Proyecto:
- **index.html**: App del Cliente (Menús y pedidos).
- **restaurante.html**: Panel de gestión para aliados.
- **domiciliario.html**: App para repartidores con rastreo.
